void TroffDisplayDriver::display ( Document *  doc)
virtual

The actual display function to be reimplemented.

This function will be called by the program to be used as the main loop for the display system. It need to be implemented.

The output of this function should be sent to STDOUT.

This function can often simply be a wrapper around DisplayDriver::display(doc, std::cout).

Parameters
docA pointer to the Document object to be displayed.
