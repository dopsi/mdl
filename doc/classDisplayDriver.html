Detailed Description

Generic display driver interface.

The DisplayDriver::display() function does need to be reimplemented to do the actual displaying.

See also
NcursesDisplayDriver and PlaintextDisplayDriver for default implementations

virtual void DisplayDriver::display ( Document *  doc)
pure virtual

The actual display function to be reimplemented.

This function will be called by the program to be used as the main loop for the display system. It need to be implemented.

The output of this function should be sent to STDOUT.

This function can often simply be a wrapper around DisplayDriver::display(doc, std::cout).

Parameters
docA pointer to the Document object to be displayed.

Implemented in NcursesDisplayDriver, TroffDisplayDriver, AnsiDisplayDriver, PlainTextDisplayDriver, HtmlDisplayDriver, LaTeXDisplayDriver, MdDisplayDriver, and RstDisplayDriver.
