1 mdl - a command line Markdown viewer
2 ====================================
3 
4 Goal
5 ----
6 
7 Build a functionnaly similar program to
8 [visit1985's mdp](http://github.com/visit1985/mdp), at least for the parser,
9 to display a Markdown file in the way *less(1)* does for text files.
10 
11 This program should be easy to extend using derivatives of the DisplayDriver
12 class to have a new frontend and derivatives of the Parser class (see the
13 status.md file for more information).
14 
15 Usage
16 -----
17 
18 * `Up`: go up
19 * `Down`: go down
20 * `Space`: go down half page
21 * `PgUp`: go up one page
22 * `PgDown`: go down one page
23 * `u`: display URL links
24 * `q`: quit program
25 
26 Building
27 --------
28 
29 This program depends on :
30 
31 * C++11
32 * ncurses
33 * Doxygen (to generate the documentation)
34 
35 To build the program, follow these steps :
36 
37 * create a `build` directory
38 * run `cmake $source_directory`
39 * run `make`
40 
41 This program cannot be installed as of now.
42 
43 To build the documentation, run `doxygen` in the source directory
44 
45 Versionning
46 -----------
47 
48 this project follows the semantic versionning guidelines provided at
49 [semver.org](http://semver.org/) with versions numbered as
50 `major.minor.revision` :
51 
52 * `major` is increased after a backwards incompatible api change.
53 * `minor` is increased after a backwards compatible api change.
54 * `revision` is increased after a change with no effect on the api.
55 
56 any version with `major` being 0 *should* not be considered stable nor
57 should its api.
58 
59 Versions history can be found in the file ChangeLog.md
