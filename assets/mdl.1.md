# mdl 1

## NAME

__corrupt__ - modify files by randomly changing bits

## SYNOPSIS

**corrupt** [**-n** *BITS*] [**--bits** *BITS*] *file*...

## DESCRIPTION

**corrupt** modifies files by toggling a randomly chosen bit.

## OPTIONS

**-n**, **--bits**=*BITS*

Set the number of bits to modify. Default is one bit.
